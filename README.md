# Core: Hogsmeade Haven

This is a project made for CORE (the tech club of DPS Dwarka) inductions.

## About the Project

### Technologies Used

**Framework**: Vue + Vite

**Styling**: TailwindCSS + DaisyUI

**Database**: Firebase (Realtime DB)

**Auth**: Firebase

### Running

Dev Server: ```
$ bun run dev

````

Build: ```
$ bun run build
````
